Step-by-step

  1. Create the app
    developers.facebook.com/apps → Create App → Other → Business → Create.
  2. Add products
    Set up Facebook Login for Business and Messenger.
  3. Messenger — no webhook required
    You do not need to configure Messenger webhooks to send broadcasts. This app links your Page to Messenger automatically when you sign in. Optional: under Messenger → Settings, confirm your Page is listed.
  4. OAuth settings
    Enable Client OAuth login and Web OAuth login. Localhost redirects are auto-allowed in Development mode.
  5. App settings → Basic
    Add localhost to App domains. Copy App ID and Secret to .env.
  6. App roles → Roles
    Add yourself as a tester while in Development mode.
  7. Restart the server and sign in again.
